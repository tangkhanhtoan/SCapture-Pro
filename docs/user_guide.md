\## 2. ⌨️ Keyboard Shortcuts / Phím tắt quan trọng



| Phím tắt            | Chức năng                    | Function                  |

|---------------------|------------------------------|---------------------------|

| ESC                 | Minimize chương trình        | Minimize window           |

| ESC (khi đang chụp) | Thoát chế độ chụp            | Cancel snipping           |

| Ctrl + C            | Sao chép (sau khi chụp)      | Copy result               |



\## 3. 🚀 Main Features / Tính năng chính



\### 3.1 📸 Capture Modes / Các chế độ chụp

\- \*\*Capture Text / Chụp Chữ (Auto)\*\*: Tự động nhận diện tiếng Việt và tiếng Anh, giữ bố cục văn bản.

\- \*\*Capture English\*\*: Chụp văn bản tiếng Anh.

\- \*\*Capture Vietnamese\*\*: Chụp văn bản tiếng Việt.

\- \*\*Capture Picture\*\*: Chụp ảnh màn hình và copy vào clipboard.



\### 3.2 🧹 Text Cleaning

\*Automatically removes bullets, icons, emotions, and table lines while keeping text layout.\*

\*Tự động xóa bullet points, icon, emotion và đường kẻ bảng, đồng thời giữ nguyên bố cục văn bản.\*



\### 3.3 📋 Result

\*All captured text or image is automatically copied to clipboard, ready to paste anywhere.\*

\*Toàn bộ văn bản hoặc ảnh sau khi chụp sẽ được tự động copy vào clipboard, sẵn sàng dán bất cứ đâu.\*
